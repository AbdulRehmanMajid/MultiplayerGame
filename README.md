# Client/Server Game

This is a Multiplayer game i developed in my free time via Unity using Netcode for Game Objects.


## Features

- Procedural Recoil
- First and third person POV
- Revive Mechanics
- Cross platform
- Low Spec Requirements
- Bullet Tracers
- Points System
- Zombies
